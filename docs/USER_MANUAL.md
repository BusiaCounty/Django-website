# Basic User Manual (Non-Technical Staff)

This guide explains how to update the public website using the Admin Dashboard.

## 1) Logging in

1. Open the Admin Dashboard: go to the site and click **Staff Login** (footer), or visit `/admin/`.
2. Enter your username and password.
3. If you forget your password, contact the ICT/Admin (Super Admin) to reset it.

## 2) Roles and responsibilities

- **Super Admin (ICT/Admin)**:
  - Creates users and assigns roles/groups
  - Full system access
- **Content Editor**:
  - Updates pages, services, news/events, notices, documents
- **Department Officer**:
  - Updates departmental items (e.g., department documents/news/notices) as assigned

## 3) Draft vs Published vs Archived

Most content has a **Status**:

- **Draft**: not visible on the public website
- **Published**: visible on the public website
- **Archived**: removed from public view (kept for records)

In Admin you can change status directly, or use the bulk actions:

- **Publish selected items**
- **Archive selected items**

## 4) Updating the main pages (Home / About / Contact)

Go to:

- **Content → Pages**

Open the page you want to edit:

- Update **Title** and **Body**
- Optional: update **Meta Title** and **Meta Description** (helps Google/SEO)
- Set **Status = Published** to make it visible

## 5) Departments

Go to:

- **Content → Departments**

Add/edit:

- Name, description
- Head of Department
- Email / phone

## 6) Services

Go to:

- **Content → Services**

Each service can be linked to a department and published when ready.

## 7) Notices & Announcements

Go to:

- **Notices → Notices**

You can add an optional **Attachment** (PDF).

## 8) News & Events

Go to:

- **Content → News items**

Use **Type** to mark an entry as **News** or **Event**.

If you upload a cover image, fill **Cover image alt text** for accessibility.

## 9) Downloads (documents)

Go to:

- **Downloads → Documents**

Upload the file (PDF/image) and set:

- Category (Forms, Reports, Policies, etc.)
- Optional department
- Status = Published

## 10) Audit trail (accountability)

Go to:

- **Audit → Activity logs**

This lists who created/updated/deleted content and from which IP/device (when available).
